# Web3Assam Winter-Cohort 2025 – Assignment 1

This is a simple full-stack project built with a Node.js + Express.js backend and a React (Vite) frontend. It reads and updates user details stored inside a JSON file.

## How to Run the Project

### 1. Backend Setup

cd backend
npm install
npm start

Backend will run on:
http://localhost:5050

---

### 2. Frontend Setup (in another terminal)

cd frontend
npm install
npm run dev

Frontend will run on:
http://localhost:5173

---

## API Routes (Backend)

GET /api/user      -> Fetch stored user data
PUT /api/user      -> Update user data (name, email, devRole)

---

## Technologies Used

- Node.js
- Express.js
- React (Vite)
- FS Module
- CORS

## Features

- Auto-creates data directory and JSON file
- Reads and updates user data
- Validates input before updating
- Clean UI using custom CSS
- Fully connected backend and frontend
